# Pathfinding Visualizer

This repository is a JavaScript Visualizer for the popular pathfinding problem found here : https://en.wikipedia.org/wiki/Pathfinding <br>
<br>
Web Interface : https://r-besson.github.io/pathfinding.js/ <br>
<br>
Algorithms Used: <br>
A* : https://en.wikipedia.org/wiki/A*_search_algorithm <br>
Dijkstra : https://en.wikipedia.org/wiki/Dijkstra%27s_algorithm <br>
DFS : https://en.wikipedia.org/wiki/Depth-first_search <br>
BFS : https://en.wikipedia.org/wiki/Breadth-first_search <br>
Greedy BFS : https://en.wikipedia.org/wiki/Best-first_search <br>
<br>
You can also select the distance type heursitic. (e.g Euclidean, Chebyshev, etc ...)
